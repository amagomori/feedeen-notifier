feedeen-notifier
====

feedeen.comの未読数をツールバー上に表示します．

## Description

ツールバーに追加されるボタンをクリックするとfeedeen.comが開かれます．feedeen-notifierは，このとき開かれたタブを監視し未読数を取得，表示します．
